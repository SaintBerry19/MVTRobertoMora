About the project:

Se generó una App tipo blog para el registro de clientes, proveedores, pedidos, materiales y facturas en una base de datos, para su futuro procesamiento en software de análisis de datos como PowerBi. Ademas se permite a los usuarios registrados ver, registrar, modificar, y eliminar entradas, todo esto mientras el usuario es reconocido por la interfaz brindadole un avatar personalizado.
